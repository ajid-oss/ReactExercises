# Exercise 6 – Greeting Title App

## 🎯 Overview

This project is part of the **Dugsiiye Mentorship Program**.
It demonstrates how to use `useState` and `useEffect` in React.

## 🚀 Features

* User enters their name
* User chooses a greeting (So Dhawow, Salaam, etc.)
* Browser title updates automatically

## 🧠 Concepts
* useState
* useEffect
